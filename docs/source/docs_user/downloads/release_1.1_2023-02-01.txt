Journal des modifications de BDOAC

Date : 01 février 2023 11 h 00 (HE)
Version : 1.1
Résumé : Version mise à jour du jeu de données sur les barrages avec une structure de données modifiée pour permettre une catégorisation plus détaillée des structures. construction_material_code a été ajouté comme nouvel attribut. Les valeurs valides pour structure_type_code et function_code ont également été modifiées pour permettre une meilleure catégorisation. La plupart des codes pour les valeurs "Inconnu" ont également été mis à jour en code 99, à l'exception de l'état de passabilité. La valeur "inconnue" pour l'état de passabilité reste le code 4. Veuillez consulter le catalogue de données sur le site de documentation du BDOAC pour obtenir la liste la plus récente des valeurs valides pour tous les attributs.
